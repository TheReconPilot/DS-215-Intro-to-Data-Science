# [DS 215] Introduction to Data Science

Repository for some course content for the DS 215 Introduction to Data Science course taught in the Aug-2025 semester by Anirban Chakraborty at the Indian Institute of Science (IISc), Bengaluru.
I am one of the Teaching Assistants (TA) for the course.

## IDS Tutorial

Notebook: [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/TheReconPilot/DS-215-Intro-to-Data-Science/blob/main/IDS_Tutorial.ipynb)


This is a basic tutorial on NumPy and scikit-learn. It covers NumPy array basics, array operations, broadcasting, reshaping and linear algebra functions. In the scikit-learn section, it covers some common patterns and gives examples of LogisticRegression and KMeans.
